# Simple Express Server

This project is a simple web server built using Node.js  and Express** with ES Modules.

The purpose of this project is to understand what a server is and how Express handles different routes, parameters, query strings, and errors.



## What is this server?

A server is a program that listens for requests from a browser and sends back responses.

This Express server responds with different text messages depending on the URL path requested.



## Technologies Used

- Node.js
- Express
- JavaScript (ES Modules)
